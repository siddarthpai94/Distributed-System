# Stanford Notes Go Examples

Small runnable Go examples inspired by the topics listed at:
https://www.scs.stanford.edu/20sp-cs244b/notes/

## Run

```bash
cd Stanford-Notes-Go-Examples
go run ./01-introduction
```

Run any topic by changing the folder name (`./02-zookeeper`, `./03-two-phase-commit`, etc.).
